想要记录一下学习过程，方便自己记录和整理，决定用hexo和github配置一下自己的博客，顺便用用git，熟悉一下操作，书看再多还是得亲自操作才知道坑在哪。

1.什么是Hexo

简单来说Hexo就是一个博客框架，可以配置多种主题，简单上手，搭建好框架后，Hexo能自动生成静态文件。

2.准备条件

2.1安装Node.js

2.1.1Node.js是什么

Node.js平台是后端运行JS代码。
安装好Node.js后npm就顺带安装好了，npm又是什么呢，是Node.js的包管理工具（package manager)。
