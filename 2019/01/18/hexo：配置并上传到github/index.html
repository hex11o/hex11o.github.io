Hexo：配置并上传到github

发表于 2019-01-18 | 分类于 FrontEnd ， 博客

1. 什么是Hexo

简单来说Hexo就是一个博客框架，可以配置多种主题，简单上手，搭建好框架后，Hexo能自动生成静态文件。

2. 准备条件

2.1 安装Node.js

  1. Node.js是什么
    Node.js平台是后端运行JS代码。
    安装好Node.js后npm就顺带安装好了，npm又是什么呢，是Node.js的包管理工具（package manager)。

2.2 安装Git

  1. Git是什么
    git是版本控制工具，为什么要版本控制及如何控制，参见git的官方文档。
